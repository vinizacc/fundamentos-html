Em html existem três formas de especificar cor: valores RGB, Hexadecimal e HSL

Valores RGB

rgb(red, green, blue)
<tag style="color: rgb(255, 0, 0)"></tag>
Cada valor pode variar de 0 a 255
Exemplo: rgb(255, 0, 0) é vermelho puro
Exemplo: rgb(0, 255, 0) é verde puro
Exemplo: rgb(0, 0, 255) é azul puro
Exemplo: rgb(255, 255, 255) é branco puro
Exemplo: rgb(0, 0, 0) é preto puro

Valores Hexadecimais

#rrggbb
<tag style="color: #FF0000"></tag>
Onde rr é o valor do vermelho, gg é o valor do verde e bb é o valor do azul
Exemplo: #FF0000 é vermelho puro
Exemplo: #00FF00 é verde puro
Exemplo: #0000FF é azul puro
Exemplo: #FFFFFF é branco puro
Exemplo: #000000 é preto puro

Valores HSL

<tag style="color: hsla(hue, saturation, lightness)"></tag>
hsl(hue, saturation, lightness)
Onde hue é o ângulo em graus, saturation é a porcentagem de saturação e lightness é a porcentagem de luminosidade
Exemplo: hsl(0, 100%, 50%) é vermelho puro
Exemplo: hsl(120, 100%, 50%) é verde puro
Exemplo: hsl(240, 100%, 50%) é azul puro

Valores HSLA

<tag style="color: hsla(hue, saturation, lightness, alpha)"></tag>
hsla(hue, saturation, lightness, alpha)
Onde hue é o ângulo em graus, saturation é a porcentagem de saturação, lightness é a porcentagem de luminosidade e alpha é a opacidade
Exemplo: hsla(0, 100%, 50%, 1) é vermelho puro
Exemplo: hsla(120, 100%, 50%, 1) é verde puro
Exemplo: hsla(240, 100%, 50%, 1) é azul puro
Exemplo: hsla(0, 100%, 50%, 0.5) é vermelho 50% opaco
