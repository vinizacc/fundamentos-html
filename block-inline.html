Elementos block e inline

O elementos contidos nas tabelas estão nessa página com as explicações, pois o contexto geral do projeto "vinizacc/fundamentos-html" não contêm todos os elementos html

Os elementos block tem a característica de "pular" a linha ápos o seu uso

Aqui são os elementos do tipo block

<adress> <article> <aside> <blockquote> <canvas> <dd>
<div> <dl> <dt> <fieldset> <figcaption> <figure>
<footer> <form> <h1>-<h6> <header> <hr> <li>
<main> <nav> <noscript> <ol> <p> <pre>
<section> <table> <tfoot> <ul> <video>


Os elementos inline tem a característica de não pular de linha ápos o seu uso, o inverso das tags block

Lista de elementos inline

<a> <abbr> <acronym> <b> <bdo> <big>
<br> <buttom> <cite> <code> <dfn> <em>
<i> <img> <input> <kbd> <label> <map>
<object> <output> <q> <samp> <script> <select>
<small> <span> <strong> <sub> <sup> <textarea>
<time> <tt> <var>
