Elemento Head

O elemento head é um elemento HTML que contém metadados sobre o documento HTML. Os metadados são informações sobre o documento, como o título, a descrição, as palavras-chave e os links para arquivos CSS e JavaScript.

O elemento head não é exibido na página da web, mas é importante para o funcionamento correto do documento HTML. Os navegadores usam as informações contidas no elemento head para exibir a página corretamente e para indexar a página nos mecanismos de busca.

  • O elemento <head> é um contêiner para metadados (dados sobre dados).
  • O elemento <head> é colocado entre as tags <html> e <body>.
  • O elemento <title> é obrigatório e define o título do documento.
  • O elemento <style> é usado para definir informações de estilo para um único documento.
  • A tag <link> é mais frequentemente usada para vincular a folhas de estilo externas.
  • O elemento <meta> é normalmente usado para especificar o conjunto de caracteres, a descrição da página, as palavras-chave, o autor do documento e as configurações da janela de visualização.
  • O elemento <script> é usado para definir JavaScripts do lado do cliente.
  • O elemento <base> especifica a URL base e/ou de destino para todas as URLs relativas em uma página.
