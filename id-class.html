Class

  • O atributo de classe do HTML especifica um ou mais nomes de classe para um elemento
  • As classes são usadas pelo CSS e JavaScript para selecionar e acessar elementos específicos
  • O atributo de classe pode ser usado em qualquer elemento HTML
  • O nome da classe é sensível a maiúsculas e minúsculas
  • Diferentes elementos HTML podem apontar para o mesmo nome de classe
  • O JavaScript pode acessar elementos com um nome de classe específico usando o método getElementsByClassName()

Id

  • O atributo id é usado para especificar um id único para um elemento HTML
  • O valor do atributo id deve ser único dentro do documento HTML
  • O atributo id é usado pelo CSS e JavaScript para estilizar/selecionar um elemento específico
  • O valor do atributo id é case-sensitive a maiúsculas e minúsculas
  • O atributo id também é usado para criar marcadores no HTML
  • O JavaScript pode acessar um elemento com um id específico usando o método getElementById()
