JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web.

JavaScript é uma linguagem de script do lado do cliente, o que significa que o código é executado no navegador do usuário, em vez de no servidor.

JavaScript é uma linguagem de programação orientada a objetos, baseada em protótipos e com suporte a funções de primeira classe.

JavaScript é uma linguagem de programação interpretada, o que significa que o código é executado linha por linha, sem a necessidade de compilação prévia.
